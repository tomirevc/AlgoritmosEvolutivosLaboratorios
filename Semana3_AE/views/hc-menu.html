CONCEPTOS

Definición

Hill Climbing es un algoritmo de búsqueda local que intenta encontrar la mejor solución moviéndose hacia estados vecinos que mejoren la solución actual.

Características

  • Greedy: Selecciona la mejor mejora inmediata.
  • Determinista o Estocástico según su implementación.
  • Sensible a máximos locales si no se aplica variación.

Aplicaciones

Optimización de horarios, planificación de tareas, juegos, y búsqueda de configuraciones óptimas.

ETAPAS

1. Estado inicial

El algoritmo comienza eligiendo un punto aleatorio en el espacio de soluciones. Este estado inicial puede ser cualquier posible solución, aunque no sea óptima, y es fundamental para comenzar la exploración.

2. Evaluación de vecinos

Se generan soluciones vecinas al estado actual. Estas variantes cercanas se comparan usando una función objetivo que mide qué tan buena es cada solución en relación con el problema que se busca resolver.

3. Movimiento

El algoritmo selecciona el vecino con la mejor evaluación y se mueve a ese nuevo estado si representa una mejora. Este proceso impulsa la búsqueda hacia soluciones más óptimas.

4. Repetición o parada

El proceso se repite desde el nuevo estado, generando y evaluando vecinos. Se detiene cuando ya no se encuentran mejoras o se alcanza un límite de iteraciones establecido.
